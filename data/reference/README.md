# NIST Framework Reference Data

## Official Document
Download the official CIS MS-ISAC NIST Cybersecurity Framework Policy Template Guide (2024) from:
    https://www.cisecurity.org/-/media/project/cisecurity/cisecurity/data/media/files/uploads/2024/08/cis-ms-isac-nist-cybersecurity-framework-policy-template-guide-2024.pdf
    
Place the PDF file in this directory as: `nist_csf_guide_2024.pdf`

## Current Reference
The `nist_framework.txt` file contains a structured summary of key NIST CSF requirements extracted from the official guide, covering:
- Core Functions (Identify, Protect, Detect, Respond, Recover)
- ISMS Requirements
- Data Privacy and Security
- Patch Management
- Risk Management
- Key Security Controls
- Compliance Requirements

This structured format is optimized for LLM context injection during policy gap analysis.
